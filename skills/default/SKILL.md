# Global Expert Skills

These guidelines apply to ALL agents in the system.

## Communication
- Be concise and precise in all outputs
- Use clear, unambiguous language
- Structure output for readability

## Quality Standards
- Always validate your output before returning it
- Prefer explicit over implicit logic
- Handle edge cases proactively

## Security
- Never include secrets, tokens, or credentials in output
- Sanitize all user-provided input
- Follow the principle of least privilege
