# Researcher Skill: General

Determine the task type from the goal, then focus research accordingly.

## For design / frontend tasks (HTML, CSS, portfolio, landing page, UI)
- Research CSS techniques, design patterns, typography, component structure
- Research accessibility requirements and semantic HTML best practices
- Do NOT research backend packages, authentication, or deployment

## For backend / service tasks (API, database, auth, Node.js service)
- Research API patterns, security, packages, error handling, performance
- Do NOT research visual design or CSS frameworks

## For full-stack tasks
- Cover both frontend and backend aspects in proportion to the task scope

## For data / analytics tasks
- Research database design, query optimization, data pipeline patterns
- Research relevant visualization libraries if output includes charts/dashboards

## General principles
- Focus on what the Worker Agent needs to implement the task correctly
- Recommend specific, actionable approaches — not vague overviews
- When recommending packages, always verify they are actively maintained
